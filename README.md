## Description

ESLint config based in recommended best practices with custom rules

## Installation

```bash
$ npm install @a2seven/eslint-react-config
```

## How to usage

1. Add @a2seven/eslint-react-config package to your .eslintrc.js file:

```javascript
module.exports = {
    extends: [
        '@a2seven/eslint-react-config',
    ],
};
```
